# Awesome-Selfhosted - Curated List of Self-Hostable Free Software

## Overview
Awesome-Selfhosted is a comprehensive, community-maintained directory of Free Software network services and web applications that can be deployed on your own servers. It serves as the definitive resource for anyone looking to replace SaaS providers with self-hosted alternatives, covering over 80 categories from analytics to wikis with hundreds of vetted software options.

## Core Features
- Extensive categorization covering 80+ software categories including analytics, automation, CMS, communication, media streaming, and more
- Each entry includes license type, programming language/platform, source code links, and demo availability where applicable
- Active community maintenance with automated checks for dead links and unmaintained projects
- Both HTML (recommended) and Markdown versions available for easy browsing
- Clear distinction between fully free software and non-free alternatives (separate list)
- Anti-features flagging system to warn about potential issues with specific software

## Quick Start Guide
1. Visit https://awesome-selfhosted.net/ for the searchable HTML version
2. Browse categories in the table of contents or use Ctrl+F to search
3. Review software entries including license, language, and demo links
4. Check the source code repository for documentation and installation guides
5. Consider hardware requirements and dependencies before deployment
6. Join the community via GitHub to contribute or request additions

## Use Cases
- Replacing cloud services (Google Drive, Dropbox, Slack) with privacy-respecting alternatives
- Building a home lab or personal server infrastructure
- Enterprise IT seeking open-source solutions for internal services
- Developers researching self-hostable components for projects
- Organizations requiring data sovereignty and compliance
- Cost reduction by eliminating SaaS subscription fees
