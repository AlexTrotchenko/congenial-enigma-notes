# Torrra - CLI Torrent Search and Download Tool

## Overview

Torrra is a Python CLI/TUI tool for searching and downloading torrents without leaving the terminal. Powered by Jackett or Prowlarr for search and Libtorrent for downloads, it offers a beautiful Textual-based interface with pause/resume support, customizable themes, and smart caching for fast searches.

## Core Features

- **Interactive TUI** — beautiful terminal UI built with Textual
- **Jackett/Prowlarr integration** — search across multiple torrent indexers
- **Direct downloads** — download via Libtorrent with pause/resume support
- **Direct magnet/torrent support** — download without searching
- **Customizable themes** — dark, light, and more
- **Smart caching** — opt-in caching for faster repeat searches
- **Cross-platform** — Linux, macOS, Windows

## Quick Start Guide

**Install:**
```bash
pipx install torrra
# or
uv tool install torrra
# or
brew install Maniacsan/torrra/torrra  # macOS
```

Also available: [AUR](https://aur.archlinux.org/packages/torrra), [Docker](https://hub.docker.com/r/stabldev/torrra), [standalone binaries](https://github.com/stabldev/torrra/releases)

**Configure Jackett:**
```bash
torrra config set indexers.jackett.url http://localhost:9117
torrra config set indexers.jackett.api_key <your_api_key>
torrra config set indexers.default jackett
```

**Usage:**
```bash
torrra                              # Interactive search (uses default indexer)
torrra search "arch linux iso"      # Direct search
torrra download "magnet:?xt=..."    # Download magnet link
torrra download "/path/to/file.torrent"  # Download .torrent file
```

## Notes & Considerations

Requires Jackett or Prowlarr running separately to provide search functionality — Torrra is the frontend/download client. The Textual TUI provides a modern terminal experience with mouse support. Caching is opt-in (`--no-cache` to bypass). Good alternative to transmission-cli or rtorrent for users who want integrated search. Remember: torrenting copyrighted content is illegal in most jurisdictions.
